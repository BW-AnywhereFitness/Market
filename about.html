What AnywhereFitness Provides to Instructors

  • Request and process payments
  • Create virtual punch passes

What AnywhereFitness Provides to Clients

  • Find out information on classes, such as
    • location
    • class size
    • start time
    • duration
  • Reschedule or cancel upcoming class reservation
